# 安卓usb摄像头监控方案demo
### 1.支持画面变化检测录像（|人脸检测录像）
### 2.支持局域网FPT服务，在局域网内可访问录像文件 
### 3.支持USB Mass（u盘），并可通过ftp访问U盘
### 4.支持存储空间监测，定期删除历史文件
